No questions unanswered!

Logorrhea is a real-time QA system for university lectures. It lets students ask questions during lectures and gives professors an overview of the class' current level of understanding.

Learn more

1. Join a channel

Use the webclient to connect to a Logorrhea server and choose a channel to join. Alternatively, use any IRC client to connect to the public server and join a channel.

2. Browse the questions

The webclient will show you a list of questions and wherether they have been answered yet. Alternatively, private message the IRC bot.

3. Ask or answer questions

Joining a question's discussion will let you answer it. You can also ask new questions. Alternatively, join the IRC sub-channels.
